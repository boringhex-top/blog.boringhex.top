## 二、多选题高频考点（136题中必考的8道）

多选题分布在以下主题中，**必须逐题过一遍**：

### 2.1 无线电管理术语（MC4 开头，约15题）

**【必背】划分 / 分配 / 指配 三连击**

| 术语 | 定义 | 记忆口诀 |
|------|------|----------|
| **划分** | 将频段列入频率划分表，供一种或多种业务使用 | "划分表" = 国家层面的大表 |
| **分配** | 将频率规定由某个/某部门在指定区域内使用 | "分配" = 分配给部门/地区 |
| **指配** | 将频率批准给**具体电台**使用 | "指配" = 指派给电台 |

**辐射 vs 发射**
- **辐射（Radiation）**：任何源的能量流以无线电波形式向外发出
  - ✅ 闪电产生的电磁波干扰是一种辐射
  - ❌ 沿电源线窜入的差模干扰、变压器磁场、电容电场 → **不是辐射**
- **发射（Emission）**：由**无线电发信电台**产生的辐射或辐射产物
  - ✅ 业余电台向周围发送的杂散产物是一种发射
  - ❌ 接收机本振辐射、医用高频电疗机、闪电 → **不是发射**

### 2.2 业余电台分类与权限（约12题）

**【必背数字】A/B/C 类功率与频段**

| 类别 | 频段 | 功率 | 操作范围 |
|------|------|------|----------|
| **A类** | 30-3000MHz | ≤25W | 只能操作VHF/UHF |
| **B类** | 全频段 | 30MHz以下≤100W，30MHz以上≤25W | 可操作HF+VHF/UHF |
| **C类** | 全频段 | 30MHz以下≤1000W，30MHz以上≤25W | 最大功率 |

**记忆口诀**：A类只有U/V段25瓦；B类短波100瓦、U/V段25瓦；C类短波1000瓦、U/V段25瓦。

### 2.3 频率划分与业余频段（约30题多选）

**【必背】业余业务作为次要业务时的义务**
- 不得对**主要业务**电台产生有害干扰
- 不得要求保护不受来自**主要业务**电台的有害干扰
- **可以**要求保护不受来自**同一业务或其他次要业务**电台的有害干扰

### 2.4 业余中继台 vs 业余信标台（约6题）

| | 业余中继台 | 业余信标台 |
|--|-----------|-----------|
| 功能 | 接收+放大转发，扩大通联范围 | 发射信标信号，验证电波传播 |
| 特点 | 双向 | 单发 |

**多选题陷阱**：信标台是"单发"不是"单收"；中继台是"放大转发"不是"衰减转发"。

### 2.5 呼号使用规则（约8题多选）

**正确使用呼号的方法**：
- 每次通信**建立及结束时**发送呼号
- 通信过程中**每10分钟内**至少发送一次
- 鼓励通过技术手段**自动发送**
- ❌ 不能用姓名、代号、别名或部分数字字母代替完整呼号

## 三、易混概念速查表

### 3.1 法规类

| 题目 | 正确选项 | 速记 |
|------|---------|------|
| 无线电管理行政法规 | 《无线电管理条例》，国务院和中央军委 | 条例=国务院+军委 |
| 业余电台管理文件 | 《业余无线电台管理办法》，工信部 | 办法=工信部 |
| 监督管理机构 | 国家和省级无线电管理机构 | 不是协会/公安/体育 |
| 业余业务定义法规 | 《无线电频率划分规定》 | 划分规定=定义 |

### 3.2 申请与执照

| 情形 | 向谁申请/办理 | 速记 |
|------|--------------|------|
| 设置使用业余电台 | 无线电管理机构 | 不是协会 |
| 终止使用注销执照 | **作出许可决定的**无线电管理机构 | 谁批的找谁销 |
| 15W以上短波/涉及国家安全 | **国家**无线电管理机构 | 大功率=国家 |
| 执照有效期届满延续 | 向核发执照的无线电管理机构申请 | 到期找原机构 |
| 注销后再次申请 | 核发已注销的呼号（不核发新的） | 呼号跟着人走 |

### 3.3 操作规范

| 情形 | 规则 | 速记 |
|------|------|------|
| 通联语言 | 普通话/英语 | 不是方言 |
| 信号报告 | RS/RST | R=可读性，S=信号强度，T=音调(CW) |
| 遇正式台/非业余台呼叫 | 不得回答/必须避让 | 业余让正式 |
| 通联内容 | 不得涉及政治/宗教/商业/个人隐私 | 四不原则 |
| 比赛/奖状通联 | 可以交换信号报告+所在地+呼号 | 三要素 |

## 五、通信程序与Q简语

### 5.1 呼叫程序

1. **CQ呼叫**：CQ CQ CQ，这里是BI1AAA，Bravo India One Alpha Alpha Alpha，呼叫任何电台，请回答
2. **指定呼叫**：BI1BBB，BI1BBB，这里是BI1AAA，呼叫
3. **回答呼叫**：BI1AAA，这里是BI1BBB，听到你信号59，请讲

### 5.2 高频Q简语

| Q简语 | 含义 | 场景 |
|-------|------|------|
| **QRA** | 你的电台名称是什么？ | 通联开始 |
| **QRZ** | 谁在呼叫我？ | 不确定谁在呼叫 |
| **QSL** | 确认收妥 / 可否寄送QSL卡片 | 最常用 |
| **QSY** | 转移到某频率 | 换频 |
| **QTH** | 你的地理位置？ | 报位置 |
| **QRP** | 降低功率（<5W） | 小功率操作 |
| **QRO** | 增加功率 | QRP的反面 |
| **QRS** | 请发慢一些 | 对方速度太快 |
| **QRQ** | 请发快一些 | 对方速度太慢 |
| **QRT** | 停止发射 / 关机 | 结束通联 |
| **QRU** | 我有事要告诉你吗？ | 通常回答"QRU"=没事 |

## 六、天线与电波传播

### 6.1 天线类型缩写

| 缩写 | 全称 | 特点 |
|------|------|------|
| **DP** | Dipole（偶极天线） | 最基本半波天线 |
| **GP** | Ground Plane（垂直接地） | 需接地网 |
| **YAGI** | 八木天线 | 定向、高增益 |
| **LP** | Log-Periodic（对数周期） | 宽频带定向 |
| **W3DZZ** | 多波段偶极 |  Trap陷波器 |
| **BEAM** | 定向天线 | 统称 |
| **VERT** | 垂直天线 | 全向 |

### 6.2 电波传播方式

| 方式 | 适用频段 | 特点 |
|------|---------|------|
| **天波（电离层反射）** | HF（短波） | 远距离通信主力 |
| **地波** | MF/LF | 沿地面传播 |
| **空间波（视距）** | VHF/UHF | 直线传播，受地形影响 |
| **散射** | VHF/UHF | 超视距通信 |

### 6.3 电离层分层

| 层次 | 高度 | 特点 |
|------|------|------|
| **D层** | 60-90km | 白天存在，吸收电波，夜间消失 |
| **E层** | 90-150km | 可反射VHF（突发E层） |
| **F1层** | 150-200km | 白天存在 |
| **F2层** | 200-400km | 最主要反射层，远距离通信主力 |

**记忆**：D层吸收、E层突发、F层反射。

## 七、电子电路基础（数字类）

### 7.1 常用公式与换算

| 公式 | 说明 |
|------|------|
| **λ = c / f** | 波长 = 光速(300,000,000 m/s) / 频率(Hz) |
| **f = c / λ** | 频率 = 光速 / 波长 |
| **dBi = dBd + 2.15** | 天线增益单位换算 |
| **dBm = 10·log₁₀(P/1mW)** | 功率电平 |

### 7.2 速算技巧

- 频率(MHz) → 波长(米)：**300 / 频率**
  - 例：14MHz → 300/14 ≈ 21米（实际20米波段）
  - 例：144MHz → 300/144 ≈ 2.08米（2米波段）
- 波长(米) → 频率(MHz)：**300 / 波长**
  - 例：40米 → 300/40 = 7.5MHz（实际7MHz）

## 八、安全防护

### 8.1 射频辐射安全

| 功率 | 安全距离 |
|------|---------|
| ≤50W (VHF/UHF) | ≥2.5米 |
| >50W (VHF/UHF) | ≥5米 |
| ≤50W (HF) | ≥5米 |
| >50W (HF) | ≥10米 |

### 8.2 避雷与接地

- **天线塔**必须安装避雷针，接地电阻 < 10Ω
- **馈线**入户处必须安装避雷器
- **设备接地**与**避雷接地**必须分开
- 雷雨天气**严禁**发射
